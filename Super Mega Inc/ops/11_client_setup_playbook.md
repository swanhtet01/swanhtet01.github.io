# Client Setup Playbook

## Goal
Set up another company on SuperMega without custom chaos.

## Step 1: Pick the starter pack

Choose one:
- `Distributor Sales Desk`
- `List Cleanup Desk`
- `Receiving Control`

Rule:
- start with the smallest pack that matches the job
- do not start with add-ons

## Step 2: Collect the current working data

Typical inputs:
- spreadsheet
- CRM export
- copied names from Google or Facebook
- WhatsApp or Viber notes
- receiving or issue log

Rule:
- use the data the team already trusts
- do not ask for perfect data first

## Step 3: Create the first working queue

For each pack:
- Sales Desk:
  - load saved companies
  - create follow-up tasks
- List Cleanup Desk:
  - import rows
  - tag usable rows
  - assign next actions
- Receiving Control:
  - import open issues
  - assign owners
  - create review queue

## Step 4: Invite the right roles

Typical roles:
- owner / founder
- manager
- operator
- clerk

Rule:
- only invite people who touch the first workflow

## Step 5: Turn on the loops

Core loops:
- `Revenue Scout`
- `List Clerk`
- `Template Clerk`
- `Task Triage`
- `Ops Watch`
- `Founder Brief`

Rule:
- do not turn on loops before the first queue is understandable

## Step 6: Expand only after usage is real

Possible add-ons:
- `Founder Brief`
- `Reply Draft`
- `Receiving Watch`
- approvals
- document intake
- client portal
- learning hub

Rule:
- expand from real usage, not ambition
