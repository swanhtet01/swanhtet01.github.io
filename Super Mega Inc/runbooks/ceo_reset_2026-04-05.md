# SuperMega CEO Reset

## What SuperMega is

SuperMega is not a broad AI platform.

SuperMega is a small workflow software company with one public trilogy:

1. Find Companies
2. Company List
3. Task List

These are simple enough to explain, demo, and sell.

## What SuperMega sells now

### 1. Find Companies

Self-serve search tool.

Outcome:
- find real companies
- keep the best ones
- draft first outreach

### 2. Sales Setup

Managed pilot built from:
- Find Companies
- Company List

Outcome:
- one clean sales list
- one follow-up rhythm
- one repeatable outreach process

### 3. Operations Setup

Managed pilot built from:
- Task List

Outcome:
- one short task queue
- fewer missed follow-ups
- one place to review blockers

## What SuperMega should stop selling for now

Do not lead with:
- AI OS
- ERP platform
- multi-agent platform
- broad manufacturing suite
- generic productivity stack

Those are internal roadmap ideas, not public offers.

## Internal agent team

These agents are internal operating loops, not the public product.

### Revenue Scout
- run saved company hunts
- clean results
- draft outreach
- seed first follow-up tasks

### List Clerk
- normalize pasted company lists
- dedupe rows
- keep stage and notes clean

### Task Triage
- convert messy updates into short task lists
- assign owner, priority, and due date

### Founder Brief
- summarize leads, tasks, issues, and conversions
- write one short daily brief

### Product Triage
- review product friction
- cut unused surfaces
- keep naming and IA consistent

## Build order

### Product
1. Make Company List shared and recoverable by default.
2. Make Task List shared and recoverable by default.
3. Keep Find Companies as the only public acquisition wedge.

### Company
1. Real booking URL
2. Work email plus company on first shared save
3. Billing
4. Product analytics
5. Support inbox

### Infrastructure
1. Cloud Run for app/API host
2. Cloud SQL Postgres for shared state
3. Cloud Tasks for background hunts and briefs
4. PostHog for product analytics
5. Resend for recovery and onboarding mail
6. Stripe for payments

## Tooling to equip the team with

Use now:
- OpenAI Responses API
- FastAPI
- React + Vite
- PostHog

Add next:
- Cloud Run
- Cloud SQL Postgres
- Cloud Tasks
- Resend
- Stripe

Add later only when the loops are already producing revenue:
- LangGraph
- PydanticAI
- richer internal agent orchestration

## Operating rule

Every new feature must close one of these loops:
- search
- save
- task
- outreach
- booking
- review

If it does not close one of those loops, cut it.
