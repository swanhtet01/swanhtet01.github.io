### Option 1: Automated Deployment (Recommended)

```bash

cd "Super Mega Inc"

# Run automated deployment
python deploy_supermega.py
```

The automated deployment will:
- ✅ Check system requirements
- ✅ Install all Python dependencies
- ✅ Set up browser drivers
- ✅ Create required directories
- ✅ Start all AI products
- ✅ Create startup scripts for future use

### Option 2: Manual Setup

```bash
# Install dependencies
pip install -r supermega_requirements.txt

# Set up browser drivers (optional)
python -m playwright install chromium

# Start services manually
python -m streamlit run browser_automation_suite.py --server.port=8503
python -m streamlit run enhanced_ai_coding_companion.py --server.port=8505
```
