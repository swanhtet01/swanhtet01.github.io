# Ai Content Generator

## Overview
Advanced AI content generation with GPT integration

## Features
- Blog writing
- Code generation
- Image creation
- Video scripts

## Technology Stack
- Python
- OpenAI
- FastAPI
- React

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python src/ai_content_generator_main.py`

## API Usage

### Start API Server
```bash
python api/ai_content_generator_api.py
```

### Available Endpoints
- `GET /` - Root endpoint
- `GET /status` - Application status
- `GET /features` - List features
- `POST /features/execute` - Execute feature
- `GET /health` - Health check

## Web Interface

Open `web/ai_content_generator_interface.html` in your browser for the professional web interface.
