# Ai Video Editor

## Overview
Intelligent video editing with automated cuts and effects

## Features
- Auto-editing
- Scene detection
- Audio sync
- Effect generation

## Technology Stack
- Python
- OpenCV
- FFmpeg
- TensorFlow

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python src/ai_video_editor_main.py`

## API Usage

### Start API Server
```bash
python api/ai_video_editor_api.py
```

### Available Endpoints
- `GET /` - Root endpoint
- `GET /status` - Application status
- `GET /features` - List features
- `POST /features/execute` - Execute feature
- `GET /health` - Health check

## Web Interface

Open `web/ai_video_editor_interface.html` in your browser for the professional web interface.
