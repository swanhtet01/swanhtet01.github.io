# Social Media Manager

## Overview
Automated social media management with AI content

## Features
- Content scheduling
- Engagement tracking
- Trend analysis
- Auto-responses

## Technology Stack
- Python
- APIs
- NLP
- React

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python src/social_media_manager_main.py`

## API Usage

### Start API Server
```bash
python api/social_media_manager_api.py
```

### Available Endpoints
- `GET /` - Root endpoint
- `GET /status` - Application status
- `GET /features` - List features
- `POST /features/execute` - Execute feature
- `GET /health` - Health check

## Web Interface

Open `web/social_media_manager_interface.html` in your browser for the professional web interface.
