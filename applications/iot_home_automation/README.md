# Iot Home Automation

## Overview
Complete IoT home automation system

## Features
- Device control
- Energy monitoring
- Security system
- Voice commands

## Technology Stack
- Python
- MQTT
- React Native
- Docker

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python src/iot_home_automation_main.py`

## API Usage

### Start API Server
```bash
python api/iot_home_automation_api.py
```

### Available Endpoints
- `GET /` - Root endpoint
- `GET /status` - Application status
- `GET /features` - List features
- `POST /features/execute` - Execute feature
- `GET /health` - Health check

## Web Interface

Open `web/iot_home_automation_interface.html` in your browser for the professional web interface.
